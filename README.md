# Todo Application (Список дел)

Простое консольное приложение на Java для управления списком задач.

## Инструкция по запуску

Для запуска приложения вам понадобятся установленные **JDK 17 (или новее)**, **Maven** и **Git**.

### 1. Клонирование репозитория

Откройте терминал и выполните команду:

```bash
git clone [https://github.com/ваш-логин/название-репозитория.git](https://github.com/ваш-логин/название-репозитория.git)
cd название-репозитория
```

### 2 Сборка JAR-файла

Если вы хотите собрать готовый исполняемый файл приложения:

```bash
mvn clean package
```

После успешной сборки готовый `.jar` файл появится в появившейся папке `target/`.

### 3. Запуск собранного JAR-файла

После того как вы выполнили сборку `mvn clean package`, программу можно запустить напрямую через Java из папки `target`:

```bash
java -jar target/название-вашего-файла.jar
```
